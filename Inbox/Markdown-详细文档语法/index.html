    Markdown 是一种轻量级的标记语言，用来编写格式简单的文档。下面是 Markdown 的详细语法：

    # 标题

    Markdown 支持六级标题，分别用 1-6 个 # 来表示：

    1
2
3
4
5
6

    # 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

    # 文本样式

    Markdown 支持粗体、斜体、删除线等样式：

    1
2
3

    **粗体**
*斜体*
~~删除线~~

    # 引用

    使用 > 号表示引用内容：

    1

    > 这是一段引用。

    可以嵌套使用多个 > 号表示不同层次的引用。

    嗡嗡嗡

    # 列表

    Markdown 支持有序列表和无序列表，有序列表使用数字加 . 号表示，无序列表使用 - 或 * 号表示：

    1
2
3
4
5

    1. 第一项
2. 第二项
- 无序列表项1
- 无序列表项2
* 无序列表项3

    可以嵌套使用多个层次。

    # 链接和图片

    可以通过 [] 和 () 来定义链接和图片的显示文本和链接地址或图片地址：

    1
2

    [链接显示文本](链接地址)
![图片显示文本](图片地址)

    # 表格

    Markdown 支持制作简单表格，使用 | 和 - 来分隔单元格和表头：

    1
2
3
4

    | 表头1 | 表头2 |
| --- | --- |
| 单元格1 | 单元格2 |
| 单元格3 | 单元格4 |

    也可以使用 : 来控制对齐方式：

    1
2
3
4

    | 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

    # 代码块

    可以通过 ` 或 包裹代码块，其中 \` 表示行内代码块， ``` 表示多行代码块：

    行内代码：print('Hello, world!')

    多行代码：

    1
2
3
4
5

    def add(x, y):
    return x + y
    
result = add(2, 3)
print(result)

    以上就是 Markdown 的详细语法。
