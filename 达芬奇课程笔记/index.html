# 一、 剪辑

# 1、变速的同方式

# 1.1 右键 - 更改片段速度

  • 波纹序列（调速后不会有空白帧）
  • 反向速度（反向播放）
  • 冻结帧（画面静止）
  • 声音校正

# 1.2 区域片段调速

  • 通过方向键调整卡帧的位置
  • 通过点击视频通道下面三角箭头选择添加添加速度点
  • 调整速度点滑杆调速

# 1.3 右键 - 变速曲线（调整曲线使加速不突兀）

# 2、升格和降格处理

# 2.1、降格

  • 智能媒体夹批量选择需要调整的视频，右键-片段属性-视频帧率调道25（60帧视频快速调整至25帧）
  • 降速卡顿调整方法（打开右上角检查器）
    • 打开retime and scaling
      • 变速处理 选项
        • 最近的帧（最近的2个帧混合播放会卡顿）
        • 帧混合（2个靠近的帧计算出中间的帧，有重影）
        • 光流（画面背景干净的情况下效果最好
  • 运动估计
    • 增加型更好

# 2.2 软件交互和工程打包

  • PR-Davinci
    • pr-文件-导出-final cut pro xml…(码率和PR需要匹配）
    • Davinci - 文件 -导入 - 时间线
    • Davinci - 渲染设置 - PR- quick-HR-HRQ -渲染队列
    • pr 打开 Davinci.xml
  • davinci - ae
    • 多个单独片段（需要设置）
      • 文件-源名称-使用独特文件名-前缀
  • 文件-媒体文件管理
    • 片段-use-时间线
    • 时间线视频选择那个导出的片段就是那个
    • 片段-copy- keeping 24(导出修改后的片段否则时源视频）
